IscAnnotation
==============

The ``IscAnnotation`` class provides access to a diagram annotation allowing annotation text to be set and cleared.

Field Values
-------------

.. list-table:: **IscAnnotation Field Values**
   :widths: 2 5 15
   :class: tight-table
   :header-rows: 1

   * - Type
     - Field Name
     - Description
   * - String
     - HTMLText
     - Gets or sets the text displayed in the annotation. A limited set of simple HTML formats is supported.
